## 4. Step 1：SFC 编译器（PHP CLI 预处理）

### 命令

```bash
php tools/sfc-compiler.php src/Calculator.vue
```

### 输入

| 输入 | 说明 |
|------|------|
| `src/Calculator.vue` | SFC 单文件组件，包含 `<template>` `<script lang="php">` `<style>` 三个块 |

### 输出

| 输出 | 说明 |
|------|------|
| `src/Calculator.gen.php` | ReactiveComponent 子类（script 块内容 + class 包装） |
| `src/CalculatorLayout_gen.php` | `WINDOW_WIDTH` / `WINDOW_HEIGHT` 常量 + `getLayout()` 函数 |

### 4步管道

| 步骤 | 功能 | 关键实现 |
|------|------|----------|
| 1. Block Extraction | 从 .vue 正则提取 template/script/style | `preg_match('#<template>(.*?)</template>#s', ...)` |
| 2. Style Resolution | CSS class → GDI BGR 颜色 | `hexToBgr()` 转换 + `borderColor()` 自动推导 |
| 3. Template Parsing | 解析自定义标签生成元素/按钮数组 | `<rect>` `<text :bind="">` `<grid>` `<btn @click="">` |
| 4. Code Generation | 输出 .gen.php 文件 | `var_export()` + heredoc 模板 |

### 模板标签参考

| 标签 | 属性 | 示例 |
|------|------|------|
| `<app>` | `title`, `width`, `height` | `<app title="VueCalc" width="328" height="420">` |
| `<rect>` | `x`, `y`, `w`, `h`, `class` | `<rect x="0" y="0" w="328" h="420" class="app-bg" />` |
| `<text>` | `x`, `y`, `:bind`, `class`, `align`, `container-w`, `container-x` | `<text y="36" :bind="display" class="display-text" align="right" container-w="320" />` |
| `<grid>` | `x`, `y`, `cols`, `rows`, `cell-w`, `cell-h`, `margin` | `<grid x="0" y="80" cols="4" rows="5" cell-w="80" cell-h="60" margin="2">` |
| `<btn>` | `row`, `col`, `label`, `class`, `@click` | `<btn row="0" col="0" label="C" class="btn-func" @click="reset" />` |

### SFC 编译器关键设计约束

- **运行在 AOT 之外**：标准 PHP CLI 脚本，不经过 AOT 编译
- **文件名**：输出文件只用 `[a-zA-Z0-9_]`，禁用点号（如 `Layout_gen.php` 而非 `Layout.gen.php`）
- **布局数据**：使用 `function getLayout(): array` 而非 `const LAYOUT = [...]`（AOT 不支持复杂 const 数组）
- **正则分隔符**：避免 `#`（与 CSS 颜色冲突），优先用 `~`

## 5. Step 2：AOT 编译器（PHP → C++ → exe）

### 命令

```bash
swoole_compiler.exe examples/vue-calc/project.yml -f
```

### project.yml 配置

```yaml
name: vue-calc          # 项目名（决定 exe 文件名）
version: 1.0.0
mode: bin               # bin = 可执行文件, ext = 扩展
no-console: false       # false = 显示控制台（调试用）
sources:
  - main.php            # 入口文件
  - ./src               # PHP 源码目录（含 .gen.php）
  - ./php-src           # C++ Stub 声明目录
  - ./cpp-src           # C++ 实现目录
```

### 关键参数

| 参数 | 说明 |
|------|------|
| `-f` | 强制全量重编译 |
| `-o` | 指定输出文件名 |
| `-j N` | 并发编译任务数 |
| `-m bin/ext` | 编译模式（bin=可执行文件） |
| `--debug-info` | 附带调试信息 |

### 编译流程

```
project.yml 配置
  ↓
prepare:  收集 sources 中所有 .php 文件      → 7 source files
  ↓
convert:  每个 .php → .cc (C++ 翻译)        → 生成 arginfo 头文件
  ↓
compile:  cl /c *.cc → *.obj (MSVC 编译)     → 11 files
  ↓
link:     *.obj + phpx.lib + php8ts.lib      → vue_calc.exe
          + user32.lib + gdi32.lib + kernel32.lib
```

### sources 配置规则

- **文件**：直接包含
- **目录**：递归遍历所有 `.php` 文件
- **相对路径**：相对于 `project.yml` 所在目录
- **ignore**：可配置排除列表（如 `vendor/autoload.php`）

## 6. 一键构建脚本

### compile_vue_calc.bat

```bat
@echo off
call "C:\Program Files\Microsoft Visual Studio\18\Community\VC\Auxiliary\Build\vcvarsall.bat" x64
cd /d F:\work\swoole_compiler

echo === Step 1: SFC Compiler (Vue → .gen.php) ===
php examples\vue-calc\tools\sfc-compiler.php examples\vue-calc\src\Calculator.vue
if %errorlevel% neq 0 (
    echo SFC Compiler failed with error %errorlevel%
    exit /b %errorlevel%
)

echo === Step 2: AOT Compilation (PHP → .exe) ===
swoole_compiler.exe examples\vue-calc\project.yml -f
echo === Exit code: %errorlevel% ===
dir /b vue_calc.exe 2>nul
dir /b vue-calc.exe 2>nul
echo === Done ===
```

### 运行方式

```bash
# 在 Git Bash / MSYS2 中：
cmd.exe //c F:\work\swoole_compiler\compile_vue_calc.bat
```

> **注意**：MSYS2/Git Bash 中 `cmd /c` 需写成 `cmd //c`（双斜杠），否则参数被 MSYS2 路径转换吃掉。

## 7. 常见构建错误速查

### 7.1 SFC 编译器错误

| 错误 | 根因 | 修复 |
|------|------|------|
| `Error: No <template> block found` | .vue 文件缺少 `<template>` 标签 | 确保三个块都存在 |
| `Unknown modifier '('` | 正则分隔符 `#` 与 CSS 颜色 `#` 冲突 | 改用 `~` 分隔符 |
| `Elements: 2 (rects + texts)` 而非 4 | `[a-z-]+` 不匹配 `:bind` | 改为 `[a-z:@-]+` |
| 按钮颜色全是默认灰色 | 颜色正则匹配失败 | 检查 hex 颜色正则分隔符 |

### 7.2 AOT 编译器错误

| 错误 | 根因 | 修复 |
|------|------|------|
| `'cl' 不是内部或外部命令` | MSVC 环境未初始化 | 先运行 `vcvarsall.bat x64` |
| `All execution code must be within a function` | `require_once` 写在顶层 | 移除；AOT 通过 sources 自动链接 |
| `Expression must be a global, non-magic constant` | `__DIR__` 在属性默认值 | 移到构造函数 |
| `error C3927/C3484: "->"` | 文件名含点号生成无效 C++ 符号 | 改下划线 `_gen.php` |
| `Cannot redeclare class X` | `require_once` 在 AOT 中重复声明 | 移除 require |
| `Undefined variable $xxx` | 使用未定义变量 | 先赋初始值（如 `= null`） |
| LAYOUT 常量未注册 | `const` 不支持复杂数组 | 改用 `function getLayout()` |
| `compile failed: extension-app_xxx.cc` | .cc 文件链接错误 | 检查项目配置和 stub 文件 |

### 7.3 链接错误

| 错误 | 根因 | 修复 |
|------|------|------|
| `unresolved external symbol` | C++ 函数未实现 | 检查 `cpp-src/*.cc` 中函数名是否正确（`php_` 前缀） |
| `LNK1104: cannot open file` | 库文件不存在 | 确认 `phpx.lib` `php8ts.lib` 路径 |

## 8. AOT 兼容性规则（必读）

> 详情参照：`swoole compiler AOT 文档/兼容性.html`

### 禁止使用的 PHP 特性

| 特性 | 说明 | 替代方案 |
|------|------|----------|
| `$$` 动态变量 | 编译器无法确定变量名 | 用数组 |
| `extract()` | 运行时创建变量 | 手动赋值 |
| `yield` / generator | 不支持生成器 | 用 Fiber/Swoole 协程 |
| 多层 `break` / `continue` | 虚拟机特性 | 用 `goto` 或 `try/catch` |
| `__get` / `__set` | 魔术方法在 C++ 中不调用 | 声明 public 属性 |
| 反射 (`ReflectionClass` 等) | 运行时动态 | 编译期确定 |
| `eval()` / `include` 动态加载 | 运行时编译 | .vue → SFC 编译器预处理 |
| 字符串含 `\0` | 与 C++ 不兼容 | 不要用 null 字节 |
| 参数数量不匹配的函数调用 | 编译器要求参数严格 | 确保参数数量正确 |
| 变量类型转换 | AOT 类型不可变 | 声明时确定类型，不改变 |

### 必须遵守的规则

1. **所有代码在函数内**：不允许顶层游离代码
2. **变量先定义后使用**：`isset($undef)` 在 AOT 中恒为 true
3. **类型不可变**：变量不能从 string 变为 object
4. **对象类型不可变**：不能 `$o = new A(); $o = new B();`
5. **文件名只用 `[a-zA-Z0-9_]`**：避免 C++ 符号名冲突
6. **全局 const 只用标量**：复杂数组用 function 返回

## 9. 为新 .vue 组件扩展构建

### 添加新组件的步骤

```
1. 编写 src/NewComponent.vue（template + script + style）
2. 运行 SFC 编译器：
   php tools/sfc-compiler.php src/NewComponent.vue
3. 生成的文件自动出现在 src/ 中，project.yml 的 ./src 自动包含
4. 在 main.php 中引入 new 新组件：
   $comp = new NewComponent('NewComponent');
5. 运行 compile_vue_calc.bat 编译
```

### 多组件项目建议

```yaml
# project.yml
name: my-app
sources:
  - main.php
  - ./src        # 包含所有 .vue 编译产物
  - ./components # 其他 .vue 组件
  - ./php-src
  - ./cpp-src
```
